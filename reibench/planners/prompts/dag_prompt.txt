===========================
DAG RULES
===========================
1. Decompose the instruction into the MINIMAL number of symbolic actions needed.
2. Each node has:
      name: <one API action instantiated with objects>
      edge: [indices of prerequisite nodes]
3. Order must be encoded via edges.
4. The last node must be "task complete".
5. Do NOT create actions that are irrelevant to the goal.
6. Do NOT invent objects or actions.

      edge: [all required nodes]
